# lad.land Website
This is my personal website, which I have open-sourced with the hopes it is interesting to somebody. You can access the site at [lad.land](https://lad.land).

This repository is hosted on [Codeberg](https://codeberg.org/ladnerlad/lad.land) and is also mirrored on [GitHub](https://github.com/ladnerlad/lad.land).

## Stack
The site is built using [Astro](https://github.com/withastro/astro), this puts most of the resource strain at build rather than the end user.

When JavaScript is used it is kept to a minimum.

Currently I use [Bun](https://github.com/oven-sh/bun), however this may change after the recent rewrite.

All posts are written in Markdown or Mdx, depending on the post. These are openly shared for public use.
